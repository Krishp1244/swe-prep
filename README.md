# ⌨️ Dev Deck

**Crack the Coding Interview** — A free, modern flashcard app for software engineering interview prep.

## 🚀 Features

- **140+ Flashcards** — Covering DSA, System Design, OOP, OS, Networking, Concurrency, and Databases
- **🤖 AI Coach** — Powered by Gemini AI for personalized interview coaching
- **📊 Progress Tracking** — Streaks, accuracy scores, and weak spot detection
- **🔐 Cloud Sync** — Sign in with Google or GitHub to save progress across devices
- **🎨 4 Premium Themes** — Midnight, Ocean, Cosmos, and Light mode
- **📖 Study Guide** — Browse all questions with expandable answers
- **💀 Weak Spots** — Auto-tracks questions you struggle with

## 🛠️ Tech Stack

- **React 19** with functional components and hooks
- **Firebase** — Auth (Google/GitHub) + Firestore for cloud save
- **Gemini AI** — Real-time AI coaching chat
- **CSS-in-JS** — Custom theme system with glassmorphism effects

## 🏃 Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm start
```

Create a `.env` file with your keys:

```env
REACT_APP_GEMINI_API_KEY=your_gemini_key
REACT_APP_FIREBASE_API_KEY=your_firebase_key
REACT_APP_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
REACT_APP_FIREBASE_PROJECT_ID=your_project_id
REACT_APP_FIREBASE_STORAGE_BUCKET=your_project.appspot.com
REACT_APP_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
REACT_APP_FIREBASE_APP_ID=your_app_id
```

## 📦 Deployment

```bash
# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```
